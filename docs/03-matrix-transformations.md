## 四种基本变换

### ① 旋转

```python
import numpy as np

theta = np.pi / 4  # 45°
R = np.array([[np.cos(theta), -np.sin(theta)],
              [np.sin(theta),  np.cos(theta)]])

v = np.array([1, 0])
print(f"[1,0] 旋转45°: {np.round(R @ v, 4)}")
```

```
[1,0] 旋转45°: [0.7071 0.7071]
```

旋转矩阵的规律：`cos` 和 `sin` 排成特定位置。det = 1（面积不变，只是转了个方向）。

### ② 缩放

```python
S = np.array([[2, 0],
              [0, 0.5]])  # x 放大 2 倍，y 缩小一半
print(f"[1,1] 缩放: {S @ np.array([1, 1])}")
```

```
[1,1] 缩放: [2.  0.5]
```

对角矩阵。det = 2 × 0.5 = 1（面积不变，形状变了）。

### ③ 剪切

```python
Sh = np.array([[1, 0.5],
               [0, 1]])   # x 方向随 y 偏移
print(f"[1,1] 剪切: {Sh @ np.array([1, 1])}")
```

```
[1,1] 剪切: [1.5 1. ]
```

矩形变成平行四边形。det = 1（面积不变，只是歪了）。

### ④ 镜像

```python
Ref = np.array([[-1, 0],
                [0, 1]])  # 左右翻转
print(f"[2,1] 镜像: {Ref @ np.array([2, 1])}")
```

```
[2,1] 镜像: [-2  1]
```

det = -1（面积不变，但方向翻了）。

**一个有趣的规律：** 旋转、剪切、镜像的 det 都是 ±1——它们不改变面积，只改变方向或形状。只有缩放会改变面积。

## 组合：多个变换叠加

1.2 已经讲过顺序很重要。这里给一个具体的例子：

```python
# 先旋转 90°，再缩放 (2, 0.5)
R90 = np.array([[0, -1], [1, 0]])   # 90° 旋转
S = np.array([[2, 0], [0, 0.5]])

v = np.array([1, 0])
print(f"先旋转再缩放: {S @ (R90 @ v)}")
print(f"先缩放再旋转: {R90 @ (S @ v)}")
```

```
先旋转再缩放: [0.  0.5]
先缩放再旋转: [0. 2.]
```

结果完全不同。组合矩阵：`S @ R` ≠ `R @ S`。

## 特征值和特征向量

### 直觉：矩阵的"骨架"

大部分向量被矩阵变换后，方向会变。但有少数特殊方向，矩阵只拉伸它们，不旋转它们。

这些特殊方向就是**特征向量**，拉伸的倍数就是**特征值**。

```python
A = np.array([[2, 1],
              [1, 2]])

# 向量 [1, 1]：方向不变，被拉伸 3 倍
v1 = np.array([1, 1])
print(f"A @ [1,1] = {A @ v1}")           # [3, 3]
print(f"3 × [1,1] = {3 * v1}")           # [3, 3]  ← 一样！

# 向量 [1, -1]：方向不变，被拉伸 1 倍（不变）
v2 = np.array([1, -1])
print(f"A @ [1,-1] = {A @ v2}")          # [1, -1]
print(f"1 × [1,-1] = {1 * v2}")          # [1, -1]  ← 一样！
```

```
A @ [1,1] = [3 3]
3 × [1,1] = [3 3]
A @ [1,-1] = [1 -1]
1 × [1,-1] = [1 -1]
```

矩阵 A 对 `[1,1]` 只是拉伸 3 倍，对 `[1,-1]` 保持不变。这两个方向是矩阵的"骨架"。

### 用橡皮布的类比

想象你在拉一块正方形的橡皮布。大部分点都会偏离原来的方向。但如果你沿着特征向量的方向拉，这些点只会离原点更近或更远，不会偏离方向。特征值告诉你拉了多少倍。

### 数学定义

特征值和特征向量的定义就一句话：

```
A @ v = λ × v
```

矩阵 A 作用在向量 v 上，结果等于 v 乘以一个常数 λ。v 就是特征向量，λ 就是特征值。

### 怎么算？（2×2 矩阵）

对于 2×2 矩阵 `[[a, b], [c, d]]`，特征值满足：

```
λ² - (a+d)×λ + (ad - bc) = 0
```

`(a+d)` 叫**迹**（trace），`(ad-bc)` 就是行列式。解这个二次方程就行。

```python
A = np.array([[2, 1],
              [1, 2]])

# 迹 = 2+2 = 4, 行列式 = 4-1 = 3
# λ² - 4λ + 3 = 0
# (λ-3)(λ-1) = 0
# λ₁ = 3, λ₂ = 1

eigenvalues, eigenvectors = np.linalg.eig(A)
print(f"特征值: {eigenvalues}")        # [3, 1]
print(f"特征向量:\n{eigenvectors}")
```

```
特征值: [3. 1.]
特征向量:
[[ 0.70710678 -0.70710678]
 [ 0.70710678  0.70710678]]
```

### 特征分解：拆出矩阵的骨架

如果一个 n×n 矩阵有 n 个线性无关的特征向量，它可以被分解为：

```
A = V @ D @ V⁻¹
```

- V = 特征向量组成的矩阵
- D = 特征值组成的对角矩阵
- V⁻¹ = V 的逆

这个分解的意思是：**先转到特征向量的坐标系，沿各轴缩放，再转回来。** 就像先把物体转到合适的角度，拉伸，再转回去。

```python
A = np.array([[2, 1], [1, 2]], dtype=float)
eigenvalues, eigenvectors = np.linalg.eig(A)

V = eigenvectors
D = np.diag(eigenvalues)
V_inv = np.linalg.inv(V)

# 重建：A = V @ D @ V⁻¹
reconstructed = V @ D @ V_inv
print(f"原始矩阵:\n{A}")
print(f"重建矩阵:\n{np.round(reconstructed, 10)}")
```

```
原始矩阵:
[[2. 1.]
 [1. 2.]]
重建矩阵:
[[2. 1.]
 [1. 2.]]
```

## 为什么特征值重要？

### PCA（主成分分析）

协方差矩阵的特征向量 = 数据变化最大的方向，特征值 = 变化的大小。按特征值排序，保留前 k 个，就是降维。

### RNN 的梯度问题

权重矩阵的特征值 > 1 → 梯度爆炸；< 1 → 梯度消失。这就是为什么 RNN 很难训练——特征值的累积效应让梯度要么飞走要么消失。

### 图神经网络

邻接矩阵的特征向量揭示图的社区结构。谱聚类用这些特征向量来做节点分组。

---

## 大模型映射

**特征值在 Transformer 里的角色：**

训练时，权重矩阵 W 的特征值如果太大，梯度会爆炸；太小，梯度会消失。Layer Normalization 和残差连接的作用之一就是稳定特征值的范围。

**PCA 在大模型里的应用：**

Embedding 维度很高（768维），但实际有效维度可能只有几百。PCA 用特征值分解找出最重要的方向，可以用来压缩 Embedding 或可视化。

## 小结

| 概念 | 一句话 |
|------|--------|
| 旋转矩阵 | 转方向，不改面积（det=1） |
| 缩放矩阵 | 拉伸/压缩，改面积 |
| 剪切矩阵 | 歪一下，不改面积（det=1） |
| 镜像矩阵 | 左右翻转，det=-1 |
| 组合 | 多个变换叠加，顺序不可调 |
| 特征向量 | 矩阵只拉伸不旋转的特殊方向 |
| 特征值 | 拉伸的倍数，矩阵的"DNA" |
| 特征分解 | `A = V @ D @ V⁻¹`，拆出骨架 |
| 迹 | 对角线元素之和 = 特征值之和 |
