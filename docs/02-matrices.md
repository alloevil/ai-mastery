## 原课程核心观点

> Every neural network is just matrix multiplication with extra steps.

每个神经网络都只是矩阵乘法加上一些额外步骤。理解了矩阵在做什么，就理解了模型在做什么。

---

## 从一个问题开始

1.1 说向量是空间中的一个点。现在我问你：**怎么把一个点搬到另一个地方？**

比如，把 `[1, 1]` 搬到 `[2, 3]`。

最简单的方式：**乘一个矩阵。**

```python
import numpy as np

M = np.array([[2, 0],
              [0, 3]])
v = np.array([1, 1])

print(M @ v)  # [2, 3]
```

```
[2 3]
```

`[1, 1]` 变成了 `[2, 3]`。矩阵 M 把它搬过去了。

但矩阵不是随便搬——它只能做**线性变换**。什么叫线性？两件事：

- 直线变完还是直线（不会变弯）
- 原点不动（不会平移）

就像你拉一块橡皮布：可以拉伸、压缩、旋转、剪切，但不能撕裂、不能折叠。

---

## 矩阵长什么样？怎么读？

一个 2×2 矩阵就是 4 个数排成方块：

```
M = | 2  0 |
    | 0  3 |
```

这 4 个数不是随便排的。**它们决定了变换的规则。**

怎么看懂它？看两列：

- 第一列 `[2, 0]` = "原来指向右边的箭头，变换后去了哪"
- 第二列 `[0, 3]` = "原来指向上方的箭头，变换后去了哪"

为什么看这两列就够了？因为平面里任何一个点，都可以用"向右走多少、向上走多少"来描述。知道了"右"和"上"这两个方向怎么变，所有点的变换就都知道了。

```python
# 基向量
right = np.array([1, 0])  # 指向右
up = np.array([0, 1])     # 指向上

print(f"变换前→变换后: 右 {right} → {M @ right}")
print(f"变换前→变换后: 上 {up} → {M @ up}")
```

```
变换前→变换后: 右 [1 0] → [2 0]   方向不变，长度变2倍
变换前→变换后: 上 [0 1] → [0 3]   方向不变，长度变3倍
```

所以这个矩阵 M 的意思是：**右边拉伸 2 倍，上方拉伸 3 倍。**

那 `[1, 1]` 这个点呢？它在"右 1、上 1"的位置。右边拉 2 倍变成 2，上方拉 3 倍变成 3，所以它去了 `[2, 3]`。

## 行列式：变换"放大了多少"

行列式（determinant，缩写 det）是一个数，从矩阵的 4 个数算出来的。

**2×2 矩阵的行列式怎么算：**

```
M = | a  b |    det(M) = a×d - b×c
    | c  d |
```

```python
M = np.array([[2, 0],
              [0, 3]])
# det = 2×3 - 0×0 = 6
print(np.linalg.det(M))  # 6.0
```

**det = 6 是什么意思？**

想象你在平面上画一个 1×1 的小正方形。经过矩阵变换后，这个正方形的面积变成了 **6**。

因为 x 方向拉伸了 2 倍，y 方向拉伸了 3 倍，面积 = 2 × 3 = 6。

**det 就是面积（或体积）的缩放倍数。**

几种情况：

```
det = 6   → 面积放大 6 倍
det = 0.5 → 面积缩小一半
det = 0   → 面积变成 0！空间被压扁了
det = -1  → 面积不变，但方向翻转了（镜像）
```

### det = 0：空间被压扁

```python
# 两个行成比例 → 空间被压扁
C = np.array([[1, 2],
              [2, 4]])  # 第二行是第一行的 2 倍
print(np.linalg.det(C))  # 0.0
```

变换后，所有点都被压到一条线上。二维空间变成了一维。**信息丢失了，无法恢复。**

这就是为什么 det = 0 的矩阵**没有逆**——就像把一张纸压成一条线，你没法把它展开回原来的纸。

## 矩阵乘法：两个变换叠加

两个矩阵相乘 = 两个变换依次执行。

```python
A = np.array([[1, 0.5],    # 剪切：右边的点随高度偏移
              [0, 1]])
B = np.array([[2, 0],      # 缩放：x 放大 2 倍，y 缩小一半
              [0, 0.5]])

v = np.array([1, 1])

# 先 B 再 A
print(f"先B再A: {A @ (B @ v)}")   # [2.25, 0.5]
# 先 A 再 B
print(f"先A再B: {B @ (A @ v)}")   # [3.0, 0.5]
```

```
先B再A: [2.25 0.5 ]
先A再B: [3.   0.5]
```

**顺序不同，结果不同。** 先拉伸再剪切 ≠ 先剪切再拉伸。

这就像穿衣服：**先穿裤子再穿鞋 ≠ 先穿鞋再穿裤子。** 步骤一样，顺序不同，结果完全不同。

所以神经网络的层顺序不能调——先过 Attention 再过 FFN，和反过来，效果完全不一样。

```python
print(f"A @ B =\n{A @ B}")
print(f"B @ A =\n{B @ A}")
```

```
A @ B =
[[2.   0.25]
 [0.   0.5 ]]
B @ A =
[[2.  1. ]
 [0.  0.5]]
```

## 逆矩阵：撤销变换

逆矩阵 `M⁻¹` 的作用：**把变换撤销掉，恢复原状。**

```python
M = np.array([[2, 1],
              [1, 3]])
M_inv = np.linalg.inv(M)

v = np.array([4, 5])
transformed = M @ v
restored = M_inv @ transformed

print(f"原始: {v}")
print(f"变换后: {transformed}")
print(f"恢复后: {restored}")
```

```
原始: [4 5]
变换后: [13 19]
恢复后: [4. 5.]
```

`M⁻¹ @ M @ v = v`。先变换，再逆变换，回到原点。

**但逆不是总存在的。** 只有 det ≠ 0 时才有逆。

det = 0 → 信息丢失 → 没法恢复 → 没有逆。

就像你把一个数字乘以 0 得到 0，你没法从 0 推回原来的数字——因为无数个数字乘以 0 都等于 0。

## 终极目标：神经网络的一层

原课程说的 `output = activation(weights @ input + bias)` 是什么意思？

```python
np.random.seed(42)

# 输入：3 个数字（比如一句话的 3 个特征）
x = np.array([0.5, 0.8, 0.2])

# 权重矩阵：2×3，把 3 个数字变成 2 个数字
W = np.random.randn(2, 3) * 0.5

# 偏置：加一个常数
b = np.array([0.1, 0.1])

# 前向传播
z = W @ x + b              # 线性变换 + 平移
a = np.maximum(0, z)       # ReLU：负数变 0

print(f"输入: {x}  (3个数字)")
print(f"变换后: {np.round(z, 3)}  (2个数字)")
print(f"激活后: {np.round(a, 3)}  (负数变0)")
```

```
输入: [0.5 0.8 0.2]  (3个数字)
变换后: [0.234 0.364]  (2个数字)
激活后: [0.234 0.364]  (负数变0)
```

逐步解释：

### 第一步：`W @ x` — 矩阵把 3 个数字变成 2 个数字

这就是矩阵变换。3 维空间的点，被映射到了 2 维空间。就像你把一个 3D 物体的影子投到 2D 墙上——维度变了，信息被压缩了。

### 第二步：`+ b` — 加一个偏置

为什么要加？没有 b，变换必须过原点。加了 b，变换可以"平移"——就像相机可以移动位置，不只是固定在原点。

### 第三步：`relu` — 负数变 0

为什么要加这个？**没有它，多层变换叠加起来还是一层变换。** 因为两个线性变换组合还是线性变换——`W2 @ (W1 @ x) = (W2 @ W1) @ x`，等于一个大矩阵。

加了 ReLU，变换变得非线性，网络才能拟合复杂的模式。就像画画——只有直线只能画几何图形，加上曲线才能画任何东西。

## 矩阵在 Transformer 里干什么？

Transformer 的每一层有两个核心操作，都是矩阵乘法：

### Attention（注意力）

```
Q = X @ W_Q    "我在找什么？"
K = X @ W_K    "我有什么？"
V = X @ W_V    "我的内容是什么？"
```

三个矩阵，把同一个输入 X 投影到三个不同的空间。Q 和 K 的点积算出"谁和谁相关"，然后用这个分数对 V 加权。

### FFN（前馈网络）

```
hidden = relu(x @ W1 + b1)   # 3 个数 → 12 个数（升维）
output = hidden @ W2 + b2    # 12 个数 → 3 个数（降维）
```

先升维再降维——在高维空间里做非线性变换，就像用放大镜看细节，看完再缩小。

## 小结

| 概念 | 一句话 |
|------|--------|
| 矩阵 | 一个变换规则，4 个数排成方块 |
| 矩阵 × 向量 | 执行变换，把点搬到新位置 |
| 行列式 det | 变换把面积放大了多少倍 |
| det = 0 | 空间被压扁，信息丢失，不可逆 |
| 矩阵乘法 | 两个变换叠加，顺序不可调 |
| 逆矩阵 | 撤销变换，恢复原状 |
| ReLU | 打破线性，让多层网络有表达力 |
| `W @ x + b` | 神经网络一层的全部 |

**1.1 说向量是数据，1.2 说矩阵是变换。数据经过变换，就是模型在"思考"。**
