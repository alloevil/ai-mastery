## 原课程核心观点

> Every neural network is just matrix multiplication with extra steps.

每个神经网络都只是矩阵乘法加上一些额外步骤。

这句话不是夸张。Transformer 的每一层、每一个 Attention Head、每一个 FFN，核心操作都是矩阵乘法。理解了矩阵在做什么，就理解了模型在做什么。

---

## 展开讲：矩阵不是"数字表格"

### 第一个直觉：矩阵是一个动作

1.1 说向量是空间中的点。那矩阵是什么？

很多人第一反应是"一堆数排成矩形"。没错，但这就像说"乐谱是一堆音符排成行"——技术上正确，但你不知道它在说什么。

**矩阵是一个动作。它把向量从一个地方搬到另一个地方。**

```python
import numpy as np

# 一个 2x2 矩阵
M = np.array([[2, 0],
              [0, 3]])

# 一个向量
v = np.array([1, 1])

# 矩阵作用到向量上
result = M @ v
print(f"变换前: {v}")    # [1, 1]
print(f"变换后: {result}") # [2, 3]
```

```
变换前: [1 1]
变换后: [2 3]
```

向量 `[1, 1]` 被搬到了 `[2, 3]`。x 方向放大了 2 倍，y 方向放大了 3 倍。

这个"搬运"就是**线性变换**——它保持直线不变、原点不变、比例关系不变。

### 第二个直觉：看基向量就够了

`[1, 0]` 和 `[0, 1]` 是二维空间的标准基向量。它们是空间的"骨架"。

矩阵的两列，就是变换后这两个基向量去的位置：

```python
M = np.array([[2, 0],
              [0, 3]])

print(f"基向量 [1,0] 变成: {M @ np.array([1, 0])}")  # [2, 0]
print(f"基向量 [0,1] 变成: {M @ np.array([0, 1])}")  # [0, 3]
```

```
基向量 [1,0] 变成: [2 0]
基向量 [0,1] 变成: [0 3]
```

**矩阵的第一列 `[2, 0]` = "原来指向右的箭头，现在指向右，长了 2 倍"。**
**矩阵的第二列 `[0, 3]` = "原来指向上方的箭头，现在指向上方，长了 3 倍"。**

知道了基向量去了哪，任何向量的变换都能算出来——因为任何向量都是基向量的线性组合。

```python
# 向量 [3, 2] = 3×[1,0] + 2×[0,1]
# 变换后 = 3×[2,0] + 2×[0,3] = [6, 6]
v = np.array([3, 2])
print(f"M @ [3,2] = {M @ v}")  # [6, 6]
```

```
M @ [3,2] = [6 6]
```

这就是矩阵乘法的本质：**不是"行列点乘"那个算法，而是"基向量变换的线性组合"这个几何事实。**

### 旋转、剪切、缩放——矩阵的三种基本动作

```python
# 旋转 45 度
theta = np.pi / 4
R = np.array([[np.cos(theta), -np.sin(theta)],
              [np.sin(theta),  np.cos(theta)]])

# 剪切（x 方向随 y 偏移）
S = np.array([[1, 0.5],
              [0, 1]])

# 缩放
Z = np.array([[2, 0],
              [0, 0.5]])

v = np.array([1, 0])

print(f"旋转45°: {R @ v}")   # [0.707, 0.707]
print(f"剪切:    {S @ v}")   # [1, 0]
print(f"缩放:    {Z @ v}")   # [2, 0]
```

```
旋转45°: [0.707 0.707]
剪切:    [1 0]
缩放:    [2 0]
```

任何线性变换——不管多复杂——都是这三种基本动作的组合。神经网络学的就是找到最好的那一组"动作"。

## 行列式：变换"缩放了多少"

行列式（determinant）是一个数字，从矩阵算出来，它告诉你：**这个变换把面积（2D）或体积（3D）缩放了多少倍。**

```python
M = np.array([[2, 0],
              [0, 3]])

det = np.linalg.det(M)
print(f"det(M) = {det}")  # 6.0
```

```
det(M) = 6.0
```

一个 1×1 的正方形，经过矩阵 M 变换后，面积变成了 6。因为 x 方向放大 2 倍，y 方向放大 3 倍，面积 = 2 × 3 = 6。

**行列式的关键含义：**

| det 值 | 意味着什么 |
|--------|-----------|
| det > 1 | 面积/体积放大 |
| 0 < det < 1 | 面积/体积缩小 |
| det = 0 | 变换把空间压扁了（降维），不可逆 |
| det < 0 | 翻转了方向（镜像） |

**det = 0 是什么情况？** 矩阵把空间压成了一个平面或一条线——信息丢失了，无法恢复。这就是为什么 det = 0 的矩阵没有逆。

```python
# 降维的例子：把 2D 压成 1D
C = np.array([[1, 2],
              [2, 4]])  # 第二行是第一行的 2 倍

print(f"det(C) = {np.linalg.det(C)}")  # 0.0
print(f"C @ [1,0] = {C @ np.array([1, 0])}")  # [1, 2]
print(f"C @ [0,1] = {C @ np.array([0, 1])}")  # [2, 4]
```

```
det(C) = 0.0
C @ [1,0] = [1 2]
C @ [0,1] = [2 4]
```

两个基向量变换后指向同一个方向——空间被压成了一条线。从 2D 到 1D，信息丢失了。

## 矩阵乘法：变换的组合

两个矩阵相乘 = 两个变换依次执行。

```python
A = np.array([[1, 0.5],    # 剪切
              [0, 1]])
B = np.array([[2, 0],      # 缩放
              [0, 0.5]])   # x 放大，y 缩小

v = np.array([1, 1])

# 先 B 再 A
result_1 = A @ (B @ v)
# 先 A 再 B
result_2 = B @ (A @ v)

print(f"先B再A: {result_1}")   # [2.25, 0.5]
print(f"先A再B: {result_2}")   # [2.  , 0.5]
```

```
先B再A: [2.25 0.5 ]
先A再B: [2.   0.5]
```

顺序不同，结果不同。先缩放再剪切 ≠ 先剪切再缩放。

```python
print(f"A @ B =\n{A @ B}")
print(f"B @ A =\n{B @ A}")

print(f"A @ B == B @ A? {np.array_equal(A @ B, B @ A)}")
```

```
A @ B =
[[2.   0.25]
 [0.   0.5 ]]
B @ A =
[[2.   1. ]
 [0.   0.5]]
A @ B == B @ A? False
```

**矩阵乘法的顺序很重要。** `A @ B ≠ B @ A`。这就是为什么神经网络的层顺序不能随意调换——先过 Attention 再过 FFN，和反过来，效果完全不同。

## 转置：翻转行和列

转置把矩阵的行变成列、列变成行。

```python
M = np.array([[1, 2, 3],
              [4, 5, 6]])

print(f"M =\n{M}")
print(f"M^T =\n{M.T}")
```

```
M =
[[1 2 3]
 [4 5 6]]
M^T =
[[1 4]
 [2 5]
 [3 6]]
```

**转置在大模型里干什么？** 反向传播。梯度从输出层传回输入层时，需要用权重矩阵的转置。这不是巧合——转置是"逆变换的近似"，在反向传播中扮演关键角色。

---

## 逆矩阵：撤销变换

逆矩阵 `A⁻¹` 的作用是：**把变换 A 撤销掉。**

```python
A = np.array([[2, 1],
              [1, 3]])

A_inv = np.linalg.inv(A)

v = np.array([4, 5])
transformed = A @ v
restored = A_inv @ transformed

print(f"原始:      {v}")
print(f"变换后:    {transformed}")
print(f"逆变换恢复: {restored}")
```

```
原始:      [4 5]
变换后:    [13 19]
逆变换恢复: [4. 5.]
```

`A⁻¹ @ A @ v = v`。先变换，再逆变换，回到原点。

**但逆不是总存在的。** 只有 det ≠ 0 时才有逆。det = 0 意味着信息丢失了，无法恢复——就像把 3D 物体压成 2D 照片，你没法从照片还原出 3D 形状。

## 神经网络的一层：终极目标

原课程说：`output = activation(weights @ input + bias)`

现在我们可以精确理解每一个部分：

```python
import numpy as np

np.random.seed(42)

# 输入：3 维向量（比如 3 个特征）
x = np.array([0.5, 0.8, 0.2])

# 权重矩阵：把 3 维映射到 2 维
W = np.random.randn(2, 3) * 0.5

# 偏置：平移
b = np.array([0.1, 0.1])

# 前向传播
z = W @ x + b        # 线性变换 + 平移
a = np.maximum(0, z)  # ReLU 激活

print(f"输入 x:  {x}        shape: {x.shape}")
print(f"权重 W:\n{W}  shape: {W.shape}")
print(f"W @ x:   {W @ x}      shape: {(W @ x).shape}")
print(f"+ b:     {z}      shape: {z.shape}")
print(f"ReLU:    {a}      shape: {a.shape}")
```

逐行拆解：

| 步骤 | 操作 | 数学含义 | 空间含义 |
|------|------|---------|---------|
| `W @ x` | 矩阵 × 向量 | 线性变换 | 从 3D 空间映射到 2D 空间 |
| `+ b` | 加偏置 | 平移 | 整个空间平移 b |
| `relu(...)` | 激活函数 | 非线性 | 把负值截断为 0 |

**`W @ x` 是核心。** 矩阵 W 的每一行是一个超平面的法向量，决定了"这个方向的特征怎么组合"。模型训练的本质就是找到最好的 W。

**为什么需要 `+ b`？** 没有 b，变换必须过原点——就像直线必须过 (0,0)。b 让变换可以"平移"，拟合更复杂的模式。

**为什么需要 `relu`？** 没有激活函数，多层线性变换叠加还是线性变换——`W2 @ W1 @ x = W_combined @ x`，等于一层。ReLU 打破线性，让网络能拟合非线性函数。

## 矩阵在 Transformer 里的具体角色

Transformer 的每一层有三个核心矩阵操作：

### 1. Attention 中的 Q、K、V

```
Q = X @ W_Q    # Query 矩阵：我在找什么？
K = X @ W_K    # Key 矩阵：我有什么？
V = X @ W_V    # Value 矩阵：我包含什么信息？
```

`W_Q`、`W_K`、`W_V` 都是学出来的矩阵。它们把同一个输入 X 投影到三个不同的空间：
- **Q 空间**：编码"我在查询什么"
- **K 空间**：编码"我能提供什么"
- **V 空间**：编码"我的实际内容"

Attention 分数 = Q 和 K 的点积（两个投影空间的相似度），然后用这个分数对 V 加权求和。

### 2. FFN 中的两层矩阵

```
hidden = relu(x @ W1 + b1)   # 升维：768 → 3072
output = hidden @ W2 + b2    # 降维：3072 → 768
```

先升维再降维——在高维空间里做非线性变换，提取更复杂的特征。

### 3. 残差连接

```
x = x + attention(x)   # 残差：原始 + 变换
x = x + ffn(x)         # 残差：原始 + 变换
```

残差连接让梯度能直接流过（不被矩阵乘法衰减），这是深层网络能训练的关键。

---

## 小结

| 原课程要点 | 大模型里的对应 |
|-----------|-------------|
| 矩阵 = 线性变换 | 每一层的权重矩阵 W 变换输入向量 |
| 矩阵的列 = 基向量变换后的位置 | W 的每一行定义一个"特征组合方式" |
| 行列式 = 面积/体积缩放因子 | 判断变换是否可逆（信息是否丢失） |
| 矩阵乘法 = 变换组合 | 多层网络依次变换，顺序不能调 |
| 逆矩阵 = 撤销变换 | 反向传播的数学基础 |
| 转置 = 翻转行列 | 反向传播中梯度的传递 |
| `W @ x + b` | 神经网络一层的前向传播 |
| ReLU 打破线性 | 让多层网络不只是"一个大矩阵" |

**一句话：矩阵是模型学到的"变换配方"。训练就是找最好的配方，推理就是用配方变换数据。**
