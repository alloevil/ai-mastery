## 从一个问题开始

你有一个模型，它有很多参数（权重）。每个参数就像一个旋钮。你需要知道：**每个旋钮往哪边拧，能让模型更准？**

如果靠猜——试一万种组合，可能永远找不到最好的。但有了导数，你**精确地知道**每个旋钮该往哪边拧、拧多少。

---

## 导数：函数的"坡度"

**直觉：** 导数就是"变化率"。如果你把 x 往右挪一点点，y 变化了多少？

```python
def f(x):
    return x ** 2

def numerical_derivative(f, x, h=1e-7):
    return (f(x + h) - f(x - h)) / (2 * h)

for x in [-2, -1, 0, 1, 2]:
    slope = numerical_derivative(f, x)
    print(f"x={x:2d}  f(x)={f(x):4d}  斜率={slope:.1f}")
```

```
x=-2  f(x)=   4  斜率=-4.0
x=-1  f(x)=   1  斜率=-2.0
x= 0  f(x)=   0  斜率= 0.0   ← 底部，最平
x= 1  f(x)=   1  斜率= 2.0
x= 2  f(x)=   4  斜率= 4.0
```

`f(x) = x²` 是一个碗的形状。在 x=0（碗底），斜率是 0——因为已经在最低点了。往左走斜率为负（在下降），往右走斜率为正（在上升）。

**几何含义：** 导数 = 切线的斜率 = 曲线在那一点的"陡峭程度"。

### 常见函数的导数

这些导数在机器学习里反复出现：

```
函数              导数              用在哪
------            ------            ------
f(x) = x²        f'(x) = 2x        损失函数（MSE）
f(x) = wx + b    f'(w) = x         线性层对权重的梯度
                  f'(b) = 1         线性层对偏置的梯度
f(x) = eˣ        f'(x) = eˣ        Softmax、Attention
f(x) = ln(x)     f'(x) = 1/x       交叉熵损失
f(x) = sigmoid   f'(x) = f(x)(1-f(x))  Sigmoid 激活函数
```

## 偏导数和梯度：多变量版本

神经网络不是只有一个参数，而是有几百万个。偏导数就是：**固定其他参数，只动这一个，损失怎么变？**

梯度 = 所有偏导数组成的向量。

```python
def f_2d(point):
    x, y = point
    return x**2 + y**2

def numerical_gradient(f, point, h=1e-7):
    grad = []
    for i in range(len(point)):
        p_plus = list(point)
        p_minus = list(point)
        p_plus[i] += h
        p_minus[i] -= h
        partial = (f(p_plus) - f(p_minus)) / (2 * h)
        grad.append(partial)
    return grad

point = [3.0, 4.0]
grad = numerical_gradient(f_2d, point)
print(f"点 {point} 的梯度: {[f'{g:.1f}' for g in grad]}")
```

```
点 [3.0, 4.0] 的梯度: ['6.0', '8.0']
```

梯度 `[6, 8]` 指向函数增长最快的方向（远离原点）。**往反方向走 `[-6, -8]`，就是下坡。**

## 梯度下降：模型学习的核心

这就是神经网络训练的全部：

```
w_new = w_old - 学习率 × 梯度
```

1. 算出损失函数对每个权重的梯度（偏导数）
2. 权重减去一小部分梯度
3. 重复

```python
x = 5.0
lr = 0.1

print(f"起始: x={x:.4f}, f(x)={x**2:.4f}")
for step in range(10):
    grad = 2 * x           # f(x)=x² 的导数是 2x
    x = x - lr * grad      # 往下坡走
    print(f"步{step+1:2d}: x={x:.4f}, f(x)={x**2:.6f}")
```

```
起始: x=5.0000, f(x)=25.0000
步 1: x=4.0000, f(x)=16.000000
步 2: x=3.2000, f(x)=10.240000
步 3: x=2.5600, f(x)=6.553600
步 4: x=2.0480, f(x)=4.194304
步 5: x=1.6384, f(x)=2.684355
步 6: x=1.3107, f(x)=1.717987
步 7: x=1.0486, f(x)=1.099512
步 8: x=0.8389, f(x)=0.703687
步 9: x=0.6711, f(x)=0.450360
步10: x=0.5369, f(x)=0.288229
```

从 x=5 出发，一步步走到 x=0（最低点）。每一步都沿着导数指的方向走一小步。

**学习率太大** → 走过头，来回震荡
**学习率太小** → 走得太慢，训练要很久

## 链式法则：反向传播的数学基础

当函数是套在一起的 `y = f(g(x))`，导数要一层层乘回去：

```
dy/dx = f'(g(x)) × g'(x)
```

```python
# 例子：y = (3x + 1)²
# 外层: f(u) = u²,   f'(u) = 2u
# 内层: g(x) = 3x+1, g'(x) = 3
# dy/dx = 2(3x+1) × 3 = 6(3x+1)

x = 2
dy_dx = 6 * (3 * x + 1)
print(f"x=2 时 dy/dx = {dy_dx}")
```

```
x=2 时 dy/dx = 42
```

**神经网络就是一串套在一起的函数：**

```
输入 → 线性变换 → 激活函数 → 线性变换 → 激活函数 → 损失
```

反向传播就是链式法则从后往前逐层应用。每一层算出自己的局部导数，乘上一层传回来的导数，传给上一层。

**这就是反向传播——整个深度学习的数学核心。**

---

## 大模型映射

### 梯度下降在 Transformer 里的角色

训练 Transformer 就是：

1. 前向传播：输入经过层层变换，算出预测值
2. 算损失：预测和真实值的差距
3. 反向传播：用链式法则算出每个权重的梯度
4. 更新权重：`w = w - lr × 梯度`
5. 重复几百万次

### 学习率调度

实际训练中，学习率不是固定的。开始大步走（探索），后面小步走（精调）。这就是学习率调度（learning rate schedule）。

### 梯度裁剪

有时梯度会突然变得很大（梯度爆炸），导致参数飞走。梯度裁剪（gradient clipping）就是给梯度设一个上限，超过就缩小。

## 小结

| 概念 | 一句话 |
|------|--------|
| 导数 | 函数的变化率 = 斜率 |
| 偏导数 | 只动一个变量时的变化率 |
| 梯度 | 所有偏导数组成的向量，指向最陡上坡方向 |
| 梯度下降 | 沿梯度反方向走，一步步到最低点 |
| 链式法则 | 复合函数的导数 = 各层导数相乘 |
| 反向传播 | 链式法则在神经网络里的应用 |
| 学习率 | 每步走多大，太大震荡太小太慢 |

**1.1 向量是数据，1.2 矩阵是变换，1.3 特征值是骨架，1.4 导数是方向。四个合起来，你就理解了"模型怎么学"。**
